Reports

Welcome to the Reports area of the Software Inventory Management System. In this area you can generate different reports which will allow you to analyze the current state of your software inventory.

Use the Inventory Listing to generate a a customizable listing of all software titles in the system based on user-selected criteria.

Use the License Exception Report to generate a listing of all software titles in the system with missing licensing information.
