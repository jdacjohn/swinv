Operating Systems List Administration

Welcome to the Operating Systems List Administration area. The Software Inventory Management System maintains information which allows you to specify the various operating systems that are supported by each of your software titles. In this area you can create, modify, and delete Operating Systems identifiers that are stored in the Software Inventory Management System..

The Add option allows you to add a new Operating System identifier to the Software Inventory Management System.

The View option allows you to view, modify and delete existing Operating System identifiers from the system.
